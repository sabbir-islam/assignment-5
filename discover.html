Question-2: What is the difference between innerHTML, innerText, and textContent?

The difference between innerHTML, innerText, and textContent are, innerHTML: Gets or sets the HTML content inside an element, including tags. innerText: Gets or sets only the visible text inside an element, ignoring hidden elements and extra spaces. textContent: Gets or sets all text inside an element, including hidden text, but treats it as plain text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements, even if they are added dynamically. Instead of attaching event listeners to multiple child elements, you attach a single event listener to a common parent.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event starts from the target element and moves up through its ancestors . It allows event handlers on parent elements to respond when a child element triggers an event. Bubbling follows the bottom-up approach.
